# 期末复习指北 - 天津科技大学版

## 📚 项目介绍

**好记性不如烂笔头**，在复习时对着书本勾勾画画，面对几百页的书本，不仅翻来翻去很可能找不到曾经认为的”重点“并且上面似乎”全是重点“。

作者从大三开始（开智晚了），期末复习时开始做笔记，复习效果立竿见影。

- 在考试前一晚只需对着手机翻看几分钟，不会因为翻书影响舍友休息。
- 在考试前几分钟，不用陷入抱着书着急忙慌翻知识点还找不到的情况。

本项目是我结合自身期末复习经验整理的「复习指北」，核心内容分为两大块：

- **复习笔记**：提炼课程核心知识点、易考点、易错点，拒绝冗余，直击重点；
- **真题/考后回忆题**：检验复习成果、考前突击。

> ✨ 注：目前覆盖的课程以笔试科目为主，后续会考虑分享大作业/课设。

## 🎯 使用指南

### 1. 下载 / 克隆项目

```
# 克隆仓库到本地
git clone https://github.com/lillee1126/TUST_Help.git
# 进入项目目录
cd TUST_help
```

### 2. 查看内容

根据学院及专业找到想要复习的课程，课程下的目录表示贡献者的考试分数，可以根据自己的情况进行选择

### 3. 推荐使用流程

⚠️注：笔记简短精炼，只起参考作用：适合**查漏补缺**和**考前突击不挂科**的同学。要想得高分，还是要好好复习

1. 先**自己复习一遍**，梳理知识点框架；
2. 找到对应**笔记**，提炼精华；
3. 做一遍**真题/模拟题**，检查能力；
4. 考前**再看**一遍笔记，巩固知识点。

## 🤝 贡献指南

我知道有很多同学都有记笔记的习惯，学校还有**最美笔记**评选比赛，如果你也想为项目添砖加瓦，欢迎：

1. 补充其他课程的复习笔记 / 试题，电子版和纸质版均可；
2. 修正现有笔记的错误、补充遗漏考点；
3. 提供正确解析。

### 贡献步骤

1. Fork 本仓库；
2. 创建分支：`git checkout -b feature/课程名称`；
3. 提交修改：`git commit -m "补充XX课程笔记/模拟题"`；
4. 推送分支：`git push origin feature/课程名称`；
5. 提交 Pull Request。

⚠️注：文件结构`学院/专业/课程/20xx-20xx 上/下学期 xx分`。分数可以为空
